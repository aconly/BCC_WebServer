Validate the Authenticity of a file / document!

A wallet or Blockchain address is not needed, as your a are only reading from the Blockchain

As a validator (e.g. employee) you get the certificate as PDF to check if it was issued by the Certifier.

When pressing "CHECK" the Hash of the file will be calculated. The Javascript on this page will read from the Smart Contract to find the Hash of the file.

When the Hash is found, all data will be returned - including the Owner Name and Public-Address of the Certifier.

As a final step you need to double-check that the Public-Address belongs to the Certifier, e.g. by checking the Certifier WebPage.

Again - the file itself will *never* be uploaded to the Blockchain.

Why Blockchain is cool here? 

  1. The Certifier does not need to build its own system! Everybody can become a certifier
  2. The Blockchain is public and can never be manipulated. This gives also security to the certificate receiver- as he can always prove he got the Certificate
  3. There is no central infrastructure that can be hacked - it is a decentral system. Only a WebPage is needed
  4. There is another use-case....THINK ABOUT IT...
